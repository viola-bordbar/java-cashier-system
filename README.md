# Java Cashier System

This project was developed as part of a programming course at KTH together with a course partner.  
It is a simple cashier system implemented in Java using basic object-oriented programming principles.  
This repository contains the final version submitted for the course.

## How to Run
Compile the source files:
```
javac -d out src/main/java/**/*.java
```

Run the program:
```
java -cp out se.kth.iv1350.processSale.startup.Main
```
